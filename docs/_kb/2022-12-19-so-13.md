How do I make the navigation tabs wrap to the next line?

---
## 1 Answers

<div class='accepted-answer-box' markdown='1'>
Add this script to the style section:

```
<style>   
    ul.nav-wizard {
        flex-flow: wrap;
        padding-right: 10px;
    }
</style>
```
